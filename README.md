# SoDogeTip Bot 

## Commands 

subject '+register' and body '+register':
    => create an account


subject '+info' and body '+info':
    => get address
   
subject '+balance' and body '+balance':
    => get your balance
  
subject '+withdraw' and body '+withdraw 100 doge to My_Address':
    => send 100 doge to My_Address
    
Tip an user : 
+/u/sodogetiptest 10 doge
